# Morningly Design System

## About

Morningly is a digital companion for Gen Z that encourages healthier mornings by gently interrupting phone scrolling with small, meaningful night-before "treats".

## About this design.md
Created using Figma Make based on Morningly high-fidelity prototype by Katherine Le Dao

---

## Logo

The Morningly logo features a cheerful sun character with a friendly face, representing morning positivity and warmth. The wordmark uses a clean, modern sans-serif font (Spartan) in dark grey (#4E4B3D).

**Logo Elements:**
- Sun character icon with smiling face
- "morningly" wordmark in lowercase
- Primary color: #4E4B3D (dark grey/black)
- Sun color: #FFE04D (yellow)

## Design Principles

1. **Friendly & Approachable:** Soft colors, rounded shapes, and a cheerful character create a welcoming experience
2. **Calming:** Pastel color palette promotes relaxation and mindfulness
3. **Playful:** Character variations and vibrant accents add personality
4. **Clear Hierarchy:** Strong typographic system ensures readability
5. **Consistent:** Unified color palette and spacing system across all touchpoints

---

## Usage Guidelines

### Do's
✓ Use the sun character to add warmth and personality  
✓ Maintain consistent spacing using the 8px grid  
✓ Use pastel colors for backgrounds and calm elements  
✓ Use vibrant colors for interactive elements and CTAs  
✓ Keep typography hierarchy clear and consistent

### Don'ts
✗ Don't use harsh, high-contrast colors  
✗ Don't mix multiple character styles  
✗ Don't use sharp corners on primary UI elements  
✗ Don't overcrowd the interface  
✗ Don't use body text smaller than 14px

---

## Accessibility

- **Contrast Ratios:** Ensure all text meets WCAG AA standards (4.5:1 for normal text, 3:1 for large text)
- **Interactive Elements:** Minimum touch target size of 48px × 48px
- **Focus States:** Clear visual indicators for keyboard navigation
- **Color Usage:** Never rely on color alone to convey information
